## BLL 业务逻辑层

## DTO 是Data Transfer Object 的简写，既数据传输对象。
是一种设计模式之间传输数据的软件应用系统。数据传输目标往往是数据访问对象从数据库中检索的数据。数据传输对象与数据交互对象或数据访问对象之间是一个不具备有任何行为除了存储和检索的数据。（访问和存取器）
<br>DTO没有任何业务行为（贫血模式）只作为数据的存储。
